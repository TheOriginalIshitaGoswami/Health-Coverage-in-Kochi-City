# Health Coverage in Kochi City: Access and Affordability Dashboard

## Overview
This project presents an interactive dashboard that analyzes healthcare access and affordability across wards in Kochi city. The dashboard combines multiple indicators related to healthcare infrastructure, service availability, cost, and accessibility to evaluate the overall health coverage within different urban wards.

The objective of this project is to identify disparities in healthcare infrastructure and affordability at the ward level, helping policymakers, researchers, and urban planners better understand gaps in urban healthcare systems.

## Project Objectives
- Analyze healthcare infrastructure distribution across wards in Kochi.
- Evaluate healthcare affordability using cost indicators.
- Examine healthcare workforce availability.
- Identify accessibility patterns based on transportation modes.
- Develop a Health Access and Affordability Index for comparative ward-level analysis.

---

## Dashboard Components

### 1. Healthcare Facility Distribution
- Community Health Centres
- Primary Health Centres
- Private Clinics
- Private Hospitals
- Sub-centres
- Veterinary Hospitals
- Other health facilities

This section visualizes the number and types of health facilities available within a ward.

### 2. Healthcare Infrastructure Indicators
Key metrics used to measure healthcare capacity:
- Average bed charges in the ward
- Number of ASHA workers per lakh population
- Average doctor–patient ratio
- Bed availability per lakh population
- Average consultation fees

These indicators help evaluate both service availability and affordability.

### 3. Transport Accessibility
The dashboard identifies the most common modes of transportation used by residents to reach healthcare facilities, such as:
- Public transport
- Private vehicles
- Walking

This helps assess physical accessibility to healthcare services.

### 4. Service Availability
Additional healthcare support services analyzed include:
- Availability of counselling services
- Availability of cashless healthcare facilities
- Presence of PM Jan Aushadhi Kendras

These indicators reflect the inclusiveness and affordability of healthcare services.

### 5. Health Access and Affordability Index
A composite index is created using multiple healthcare indicators to measure overall health coverage across wards.

The index is visualized using a ward-level map that highlights areas with:
- Higher healthcare accessibility
- Moderate coverage
- Lower access to healthcare services

This allows easy comparison between wards.
